# 米豆的博客

如果你正好刷到了我的这个博客仓库，也挺有缘分的，不嫌啰嗦可以看看下面的博客历程  

## 前言

最开始的博客是部署到了树莓派上的，使用的Ghost博客，那会只有树莓派  
后面有了9.9的阿里云，搭建了自己的wordpress博客  
wordpress博客不太灵活，需要去页面写博客，导入markdown格式不正确且图片也有问题  
之后就选择了hexo博客，但可惜那会儿开始就不写博客了，改为纯粹的笔记记录了  
现在是2026年，我们重新开始吧

## 主题、插件  

### 主题：NotrhStar  

这个主题是我用AI生成的，流程是用Pencil仿写自己喜欢的网站的风格，我不是设计出身，没法让pencil设计一套设计原则，配色这些，但是你指定好生成的规则，脑海中有一个自己想要的样子我觉得就可以了  
博客的功能，功能需要考虑到主页布局，主页布局包括英雄区，标签分类块，每一篇文章的摘要卡片，顶部导航栏。博客页需要考虑标题，图片箱，代码库，正文排版这些  
同时，我还有一个记录笔记的notes仓库，我作为博客的子仓库引入进来了，这样也可以一并生成博客页面

### 插件  

#### hexo-midou-img-collect

这个插件是用于notes这个仓库的，因为我写的笔记会有很多图片，都是存放在子仓库中对应分类的和笔记同名的文件夹中，需要一个图片收集插件，把笔记仓库中的图片放到博客对应的图片文件夹下，生成博客时才能正确的发布图片

#### hexo-midou-img-lightroom

这个插件是图片箱，默认的hexo图片是静态的，不可点击的，这个插件给图片加入了缩放功能，同一篇文章中图片画廊这些功能
